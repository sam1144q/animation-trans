CSS Selectors

CSS selectors are used to target HTML elements to apply styles to them. There are different types of selectors, such as:

  • Element Selector: Selects elements by their tag name (e.g., `

    `, `

    `, etc.).

  • Class Selector: Selects elements with a specific class attribute (e.g., `.class-name`).
  • ID Selector: Selects an element with a specific ID (e.g., `#id-name`).
  • Attribute Selector: Selects elements based on their attributes (e.g., `[type="text"]`).

CSS Box Model

The CSS Box Model is essential for understanding how elements are sized and spaced. It consists of:

  • Content: The actual content inside the element (text, images, etc.).
  • Padding: Space between the content and the border, inside the element.
  • Border: The boundary surrounding the padding and content.
  • Margin: Space outside the border, separating the element from others.

CSS Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layouts. It works by distributing space along a row or column. Key properties include:

  • display: flex; - Enables the flexbox layout on a container.
  • justify-content: Aligns items horizontally within a container.
  • align-items: Aligns items vertically within a container.
  • flex-direction: Defines the direction of the flex items (row or column).

CSS Grid Layout

CSS Grid Layout provides a two-dimensional layout system, making it easy to design complex layouts. You can define rows and columns, and position items accordingly. Key properties include:

  • display: grid; - Enables the grid layout on a container.
  • grid-template-columns: Defines the number and size of columns.
  • grid-template-rows: Defines the number and size of rows.
  • grid-gap: Defines the space between rows and columns.

CSS Animations

CSS Animations allow you to create smooth transitions and movement for elements on a webpage. You can define keyframes for the animation to change styles at specific points in time. Example properties include:

  • @keyframes: Defines the stages of an animation.
  • animation-name: Specifies the name of the animation to be applied.
  • animation-duration: Defines how long the animation takes to complete.
  • animation-timing-function: Defines the speed curve of the animation.
